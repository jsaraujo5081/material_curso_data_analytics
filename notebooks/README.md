# Contenido del Directorio Notebooks

## Fundamentos de R y Análisis de Datos

### Sprint 1 - R Básico I
*(Tiempo estimado: 6 horas)*

* **S01_R_Basico_I_Sesiones**: Ejercicios a realizar de manera guiada sobre manejo de variables, gestión de errores, manipulación de cadenas de texto y manejo de vectores atómicos y listas.
* **S01_R_Basico_I_Ejercicios**: Ejercicios adicionales para ser resueltos de manera autónoma, a fin de alcanzar mayor destreza y seguridad sobre la sintaxis del lenguaje de programación R.

### Sprint 2 - R Básico II
*(Tiempo estimado: 7 horas)*

* **S02_R_Basico_II_Sesiones**: Ejercicios a realizar de manera guiada sobre manejo de diccionarios, creación de funciones, uso de bucles y uso de condicionales.
* **S02_R_Basico_II_Ejercicios**: Ejercicios adicionales para ser resueltos de manera autónoma, a fin de alcanzar mayor destreza y seguridad sobre la sintaxis del lenguaje de programación R.

### Sprint 3 - Manipulación de Datos I
*(Tiempo estimado: 7 horas)*

* **S03_Manipulacion_Datos_I_Sesiones**: Caso de estudio sobre el ranking de selecciones masculinas de la FIFA, para practicar funciones de manipulación de datos con Tidyverse.
* **S03_Manipulacion_Datos_I_Ejercicios**: Ejercicios adicionales sobre datos de comercialización de vehículos, a fin de repasar funciones de Tidyverse.

### Sprint 4 - Manipulación de Datos II
*(Tiempo estimado: 7 horas)*

* **S04_Manipulacion_Datos_II_Sesiones**: Caso de estudio sobre ventas de un operador turístico, para practicar funciones de manipulación de datos con Tidyverse y visualizaciones con ggplot.
* **S04_Manipulacion_Datos_II_Ejercicios**: Ejercicios adicionales sobre datos simulados para repasar funciones de Tidyverse y ggplot.

## Análisis Exploratorio de Datos

### Sprint 5 - Análisis Exploratorio Descriptivo
*(Tiempo estimado: 8 horas)*

* **S05_Analisis_Descriptivo_Sesiones**: Caso de estudio sobre indicadores macoeconómicos por país provistos por el Banco Mundial, para practicar conceptos estadísticos asociados a descripción y visualización de datos.
* **S05_Analisis_Descriptivo_Ejercicios**: Ejercicios adicionales para una mayor comprensión de teoría estadística vinculada a Ley de los grandes números y el Teorema del límite central.

### Sprint 6 - Análisis Exploratorio Inferencial
*(Tiempo estimado: 8 horas)*

* **S06_Analisis_Inferencial_Sesiones**: Caso de estudio sobre rendimiento de empleados en un call-center, para practicar conceptos estadísticos asociados a inferencia mediante test de hipótesis e intervalos de confianza para la ejecución de pruebas A/B.
* **S06_Analisis_Inferencial_Ejercicios**: Caso de estudio sobre cambios en tasas de conversión en el proceso de venta de una aplicación de pedidos de comida, para practicar conceptos estadísticos asociados a inferencia, pruebas A/B y análisis de embudos.

## Comunicación de Resultados

### Sprint 7 - Visualización Interactiva
*(Tiempo estimado: 7 horas)*

* **S07_Visualizacion_Interactiva_Sesiones**: Caso de estudio sobre generación de insumos para reporte de resultados comerciales en un restaurante, para practicar funcionalidades en el uso de plotly como herramienta de graficación interactiva.
* **S07_Visualizacion_Interactiva_Ejercicios**: Ejercicios adicionales para la creación de controladores y animaciones con plotly, así como la generacion de archivos html para compartir visualizaciones interactivas.
