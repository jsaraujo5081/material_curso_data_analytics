# Contenido del Directorio Notebooks

## Fundamentos de R y Análisis de Datos

### Sprint 1 - R Basico I
*(Tiempo estimado: 6 horas)*

* **S01_R_Basico_I_Sesiones**: Ejercicios a realizar de manera guiada sobre manejo de variables, gestión de errores, manipulación de cadenas de texto y manejo de vectores atómicos y listas.
* **S01_R_Basico_I_Ejercicios**: Ejercicios adicionales para ser resueltos de manera autónoma, a fin de alcanzar mayor destreza y seguridad sobre la sintaxis del lenguaje de programación R.

### Sprint 2 - R Basico II
*(Tiempo estimado: 7 horas)*

* **S02_R_Basico_II_Sesiones**: Ejercicios a realizar de manera guiada sobre manejo de diccionarios, creación de funciones, uso de bucles y uso de condicionales.
* **S02_R_Basico_II_Ejercicios**: Ejercicios adicionales para ser resueltos de manera autónoma, a fin de alcanzar mayor destreza y seguridad sobre la sintaxis del lenguaje de programación R.
